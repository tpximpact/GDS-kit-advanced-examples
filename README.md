# GDS-kit-advanced-examples
A setup of the prototyping kit providing annotated examples of advanced features, for use by designers

Steps to install:

1. Install the app:
`npm install`

2. Intall the filters:
`npm install @x-govuk/govuk-prototype-filters`

3. Run the app:
`npm run dev`

4. Open the app:
http://localhost:3000/
